建站准备

发表于 2020-06-12 更新于 2020-06-13 分类于 建站 阅读次数： Valine：
本文字数： 2.4k 阅读时长 ≈ 6 分钟

选择域名

域名建议跟内容主题一致，这样不但更容易记忆而且用户信任感会更高，国内的域名平台建议选择：

  • 腾讯云的DNSPod或者
  • 阿里云的万网

都有丰富的域名可供选择，但朗朗上口、双拼或者短域名都是稀缺资源。

如果计划选择国内的云主机，需要进行域名备案，阿里云和腾讯云均有域名备案平台，购买完域名和云主机后，可以直接进行备案。既然是个人网站，域名实名认证和备案主体都选个人就OK，方便快捷审核快。特别提醒：域名备案的备案服务号需要有服务器才可以生成，并且要求服务器购买不低于3个月，请各位知悉。

主机/服务器

Web服务器，用来存放网站程序、图片等。

之前很多人建站选择购买虚拟主机，也就是所谓的空间，现在云计算时代，空间所能承载的业务量太小，而云服务器成为很多建站用户的首选；国内前二的云计算平台阿里云、腾讯云都是不错的选择，并且都对新用户推出了力度很大的优惠活动，例如腾讯云推出的限时秒杀活动，1核2G1M带宽低至95元/年，除了这个配置外，还推出诸如2核4G、4核8G等适合中小企业应用的机型，活动链接可参考：腾讯云618

阿里云的新用户活动也推出了优惠机型，但要先提醒大家，一定不要选择突发性能实例，该类型实例CPU仅有10或者20%，CPU受限会严重影响服务器的性能，从而影响使用体验；阿里云618年中大促百亿补贴活动于近日开启，老用户新购即可享受满减活动，新用户即可享受0.9折购机活动，活动力度前所未有，阿里云的活动链接可参考：阿里云618

服务器和空间的区别：

  • 服务器适合搭建很多网站，便于管理
  • 空间，搭建1-2个网站，100-500一年，大概150

关于备案

  • 国外、香港的空间就不需要备案。
  • 国内的就需要备案，传要求的资料即可。

建站程序

如果需要主题模板，我们可以去织梦猫或者百度搜索织梦CMS模板、wordpress模板即可。
WordPress·Hexo·Ghost·Discuz

博客程序

  • Hexo,可以说是目前最流行的博客框架了，基于Nodejs
  • wordpress，目前全球也是中国使用最多的博客程序。官方网站是 http://cn.wordpress.org，基于PHP语言。
  • Z-blog，国人制作的的功能强大的博客程序。官方网站是：http://www.rainbowsoft.org/，是基于ASP语言。当然还有如emlog等等。
  • Hugo，速度快，灵活，比如陈计节个人博客
  • laravel，比如张仿松个人博客

门户系统

  • dedecms，也就是织梦，目前国内使用最多的门户系统。网站是：http://www.dedecms.com/，基于PHP语言。有一定安全漏洞，经过处理也是可以的。在织梦官网下载建站程序，有2个版本：
    • UTF8：英文的，想要国外也能访问用这个，当然中文的也可以
    • GBK：简体中文，国外就看不了
  • phpcms，另一款很受欢迎的门户系统，网站是：http://www.phpcms.com，也是基于PHP语言。还有诸如：excms等等我就不一一举例了。

论坛系统

  • discuz，国内使用最多的论坛系统。网站是：https://www.discuz.net/forum.php，基于PHP语言。比如踏花行。
  • phpwind，另一款用户较多的论坛系统。网站是：http://www.phpwind.com，也是基于PHP语言。

如何远程管理网站空间

  1. 下载FlashFxp，FTP工具
  2. 用地址、账号、密码登录（去主机购买网站找）
  • /: FTP根目录，禁止写入，只可读取
  • htdocs/:是存放网站文件的根目录（php,html,htm,css,js等）
  • htdocs/logreport/: 访问统计报告存放目录，不占用空间，只可读取
  • myfolder/: 此文件夹中的内容不会被web用户看到，可以临时存放一些不愿公开的内容，比如站点备份、数据库备份等
  • ftplogs/: ftp访问日志存放位置，只读
  • wwwlogs/: www访问日志存放位置，只读
  • backup/: 系统自动生成的数据库备份的存放位置，只读
  1. 空间想要解压，可以在主机的管理平台（控制面板）进行解压、压缩等操作

域名与空间绑定

在域名控制台

  1. 在空间的控制台，添加解析
  2. 如果主机有IP，用A记录，记录值填ip
  3. 主机记录一般填www，如果填@，就是空前缀
  4. 还有一种是CNAME，记录值填写临时域名，这种适合没有给出IP的空间

在空间端也需要绑定域名，这边直接填入域名九型了，比较简单。

建站程序怎么上传到空间

  1. 把建站程序解压出来
  2. 以织梦为例，将uploads文件夹压缩
  3. 用FTP工具将这个uploads.zip上传到空间
  4. 在主机管理平台进行解压缩
  5. 解压后的所有文件都在htdocs/根目录下才是对的

安装建站程序

  1. 访问绑定的域名就可以安装建站程序了
  2. 填入数据库信息，从主机的管理平台获取，主机填数据库地址
  3. 按照用户、密码、数据表前缀（可以自定义）、名称填写
  4. 设置管理员密码
  5. 可以设置PHP的版本

网站后台功能

  1. 域名/dede/可以进入网站后台
  2. 核心=文章 模块=插件 生成=生成网页
